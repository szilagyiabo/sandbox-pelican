### Requirements
Python 2.7.10

### Installation
It is required to run this in [virtualenv](http://docs.python-guide.org/en/latest/dev/virtualenvs/).

If you use virtualenv run the following:
```sh
$ virtualenv venv
$ . venv/bin/activate
```

Install Sandbox-Pelican using pip:
```sh
$ pip install --editable .
```

### Usage
Help page:
```sh
$ sandbox --help
```
